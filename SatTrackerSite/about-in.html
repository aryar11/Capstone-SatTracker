About SatTracker

This website was created as a ECEN Capstone project by Kyler Holley, Arya Rahmanian, and Stuart Kinzel. It's purpose is to provide a way for people to track satellites using an interactive globe. This project was inspired by the SWAT tool, a satellite tracking software that meant for government use.
